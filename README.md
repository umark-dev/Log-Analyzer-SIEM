# Log Analyzer (Mini SIEM)

## Description
This project simulates a basic SIEM system that analyzes log files and detects suspicious activities such as brute force attacks and unauthorized admin access.

## Features
- Detects multiple failed login attempts
- Identifies suspicious admin access
- Generates alert reports

## Technologies
- Python
- Log Analysis
- SOC Concepts

## How to Run
python analyzer.py

## Output
Alerts are saved in alerts.txt
